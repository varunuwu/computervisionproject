#  Student Performance Predictor

##  Problem Statement
Students' academic performance is influenced by multiple factors such as study time, attendance, and sleep patterns. However, it is often difficult to estimate how these factors collectively impact final marks.

---

##  Objective
The objective of this project is to build a machine learning model that predicts student marks based on:
- Study hours
- Attendance percentage
- Sleep hours

---

##  Features
- Data preprocessing and visualization
- Linear Regression model for prediction
- Evaluation using MSE and R² score
- User input-based prediction
- Graphical representation of results

---

##  Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

---

##  Dataset
This project uses a **synthetic dataset** created for demonstration purposes.  

### Features:
- `study_hours` → Number of hours studied per day  
- `attendance` → Attendance percentage  
- `sleep_hours` → Average sleep hours  
- `marks` → Final marks scored  

---

##  How It Works
1. Data is created and loaded into a DataFrame  
2. Features and target variables are defined  
3. Data is split into training and testing sets  
4. A Linear Regression model is trained  
5. Predictions are made and evaluated  
6. Results are visualized
